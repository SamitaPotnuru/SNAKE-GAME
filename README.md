# Snake Game

A classic Snake game built as a standalone HTML file — no dependencies, no build step.


## 🕹️ Controls

| Input | Action |
|-------|--------|
| `↑` `↓` `←` `→` | Arrow keys to move |
| `W` `A` `S` `D` | Alternative movement keys |
| On-screen buttons | Mobile controls (touch devices) |
| Swipe on canvas | Touch swipe controls |

## ✨ Features

- 🎨 Smooth neon-styled graphics on HTML5 Canvas
- 📊 Score tracking with localStorage high score
- 🌀 Wrap-around walls (no death on edges)
- 📱 Responsive with mobile touch & swipe support
- 📋 In-game instructions box
- 🚀 Zero dependencies — single HTML file

## 🛠️ Tech

Pure **HTML + CSS + JavaScript**. No frameworks, no build tools, no npm packages.

## 🚀 Getting Started

1. Clone or download this repo
2. Open `index.html` in any modern browser
3. Press an arrow key or tap to start playing!
